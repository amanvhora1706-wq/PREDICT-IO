## API Configuration
This project uses third-party APIs (Alpha Vantage, RapidAPI).
API keys are intentionally excluded for security reasons.

To enable live data:
1. Create a RapidAPI account
2. Subscribe to the required APIs
3. Replace `YOUR_API_KEY_HERE` with your own key
